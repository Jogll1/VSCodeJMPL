# JMPL for VSCode

## Features
- Syntax highlighting
- Code snippets
- Symbol insertion (using `\`)
- Auto indent

### Snippets
This is the full list of code snippets:

| Shortcut | Code  | Description |
|------|---------|------|
| {x} | {x \| x ∈ S} | A set-builder. |
| for | for x ∈ S do | A for loop. |
